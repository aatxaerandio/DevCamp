Flex-basis --> gives us a starting point (SIZE) to the element. If the top block is 400px; and then 200px, and the third 100px. It will take the rates and no matter the screen sizwe will maintain that sizes ratio. You use the same values for width and height. flex-basis: you give it a start size and it mantains the ratio even when you change screen size flex-grow: tells to each element how much to grow in respective to the rest of the flex-grow element -> element with bigger number is going to grow X times faster than the others flex-shrink: opposite of flex-grow -> element with bigger number is going to shrink X times faster than the others
